# 法院起诉素材智能体 v2.0

## 📋 项目概述

一款智能法律辅助工具，帮助用户快速生成规范的诉讼文书，适用于个人和中小企业处理常见民商事纠纷。

## ✨ v2.0 新特性

### 🎨 界面全面升级
- **专业法律风格配色**：深蓝主色调 + 金色强调色，符合法律文书庄重气质
- **顶部进度条**：可视化显示 4 个步骤的完成状态
- **响应式双栏布局**：左侧侧边栏导航 + 右侧操作区
- **现代化组件设计**：卡片式布局、圆角设计、平滑动画过渡

### 🤖 AI 智能辅助填写
- **案情描述分析**：输入自然语言描述案情，AI 自动分析案由
- **智能生成诉讼请求**：根据案情自动生成规范的诉讼请求内容
- **智能生成事实理由**：AI 撰写符合法律格式的事实与理由文本
- 支持借款纠纷、合同纠纷、劳动合同纠纷、离婚纠纷等多种案由

### 💾 数据本地持久化
- **自动保存**：所有表单内容实时保存到 localStorage
- **多案件管理**：支持创建、切换、删除多个案件
- **历史记录**：左侧抽屉显示案件列表及修改时间
- **数据不丢失**：刷新页面后自动恢复上次编辑的内容

### 👥 多被告支持
- **动态添加被告**：支持添加多个被告人
- **独立管理**：每个被告的信息独立填写和管理
- **类型切换**：支持自然人/法人类型切换
- **一键删除**：多余的被告可随时删除

### 📂 证据清单优化
- **自动页数识别**：智能识别 PDF 页数，其他格式按文件大小估算
- **连续页码编排**：上传多个证据后自动生成连续页码范围
- **拖拽排序**：支持拖拽行调整证据顺序
- **智能分类**：根据文件名自动猜测证据类型（书证、物证、视听资料等）
- **多种导出**：Excel（CSV）和 PDF 打印格式

## 🎯 功能模块

### 1. 起诉状生成（Panel 1）
- 填写原告/被告信息（支持自然人、法人类型）
- 选择案由（13 类预置案由 + 自定义）
- 填写诉讼请求、事实与理由
- 一键生成符合法院格式的起诉状
- 支持打印/导出 PDF、复制文本

### 2. 身份信息文件（Panel 2）
- 上传原告/被告身份证件图片（JPG/PNG/PDF）
- 模拟 OCR 识别（从起诉状同步信息）
- 手动核对/补充身份信息
- 生成当事人信息确认书

### 3. 证据清单（Panel 3）
- 批量上传证据文件（PDF/JPG/PNG/DOCX/XLSX）
- 自动识别页数并连续编排页码
- 拖拽行调整证据顺序
- 支持手动添加/删除/编辑证据
- 导出 Excel 证据清单、打印 PDF

### 4. 案件检索分析（Panel 4）
- 根据案由、地区、年份等条件检索类似案件
- 内置示例案例数据
- 显示胜诉率统计
- AI 生成诉讼策略建议

## ⚙️ 技术栈

- **前端框架**：原生 HTML5 + CSS3 + Vanilla JavaScript（无框架依赖）
- **样式方案**：CSS 自定义属性 + Flexbox + Grid
- **存储方案**：浏览器 localStorage
- **字体**：Noto Serif SC（衬线体用于预览）+ Inter（无衬线体用于界面）
- **文件处理**：FileReader API、Blob、URL.createObjectURL

## 🔒 数据隐私

- 所有数据保存在**浏览器本地**，不上传到任何服务器
- localStorage 清除后数据会丢失，请注意备份重要内容
- 建议定期导出生成的文档作为备份

## ⚠️ 免责声明

本系统仅供**学习和研究参考**使用，生成的内容**不构成法律意见**。

实际提交法院前，请务必：
1. 咨询专业律师
2. 仔细核对所有信息的准确性
3. 根据案件具体情况调整内容
